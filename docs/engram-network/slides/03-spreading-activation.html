Spreading Activation Retrieval

How the brain finds memories — not by searching, but by resonance

❌ How every AI does it today
1. User asks a question
2. Embed the question
3. Find top-k nearest vectors
4. Stuff them into context
"What should I have for dinner?" → finds memories about dinner, food, restaurants. Misses: your allergy conversation, that you're fasting today, your partner's birthday tonight.
✅ How the Engram Network does it
1. User asks a question
2. Activate seed engrams (embedding similarity)
3. Activation spreads through graph edges
4. Multi-path convergence amplifies relevant memories
"dinner?" → dinner → restaurant → allergy conversation → also: today → fasting → also: date → partner's birthday
The Algorithm: 4 Steps
1
Seed Activation
Embed the query. Find top-N engrams by cosine similarity (just like today's RAG). But these are seeds, not the final result. Each gets activation = similarity_score.
2
Spread Through Edges
Each active engram passes activation to neighbors:
neighbor.activation += source.activation × edge.weight × decay_factor
Decay factor (0.5-0.8) prevents activation from spreading infinitely. Typically 2-3 hops.
3
Convergent Amplification
Engrams reached by multiple independent paths get boosted. "Birthday" is reached via dinner → tonight → birthday AND dinner → restaurant → birthday — double activation. This is the key insight: convergence = relevance.
🎂 birthday
2 paths → HIGH activation
vs
🍽️ meal plan
1 path, weak edge → low
4
Collect & Rank
After N hops (configurable, default 3), collect all engrams above activation threshold. Rank by: final_activation × importance × recency_boost. Top-K become memory context for the LLM.
⚡ Performance
Seed: ~5ms (pgvector HNSW)
Spread: ~10-50ms (graph traversal in SQL or cached in Redis)
Total: <100ms for personal scale (10k-100k engrams)
🎛️ Tuning Knobs
seed_count: initial seeds (default 10)
max_hops: spread depth (default 3)
decay_factor: per-hop decay (default 0.6)
activation_threshold: min to surface (default 0.1)
max_results: final context size (default 20)
🧪 Hybrid Fallback
If the graph is sparse (new user, few engrams), fall back to current hybrid search (vector + keyword). As the graph grows, spreading activation naturally takes over.
