🔮 Spreading Activation
Query activates seed nodes → activation flows through edges → converging paths amplify → most activated = most relevant
Not similarity search.
Finds "coffee → morning standup" even though they're semantically unrelated.
🧠 Reconstruction
Fragments reassembled into coherent memory, influenced by current context. Same fragments → different recall depending on what you're thinking about.
Not verbatim retrieval.
Like human memory — reconstructed, not replayed.
⚙️ Working Memory Gate
Context window = managed working memory. Attention gate decides what enters/exits. Active refresh beats context rot.
Not a dumb buffer.
Important things stay. Stale things evict. No rot.
🌙 Consolidation Daemon ("Sleep Cycle")
Background process: replays episodes → extracts patterns → forms schemas → strengthens important edges → prunes weak ones → resolves contradictions
Runs on idle / scheduled intervals
🧬 Neural Memory Router
Small NN (not the LLM) trained on YOUR memory access patterns. Learns personal associations. Predicts what you need before you ask.
Phase 2 — trains as you use Nova
